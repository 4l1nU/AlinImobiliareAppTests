# AlinImobiliare Selenium Tests

Automated tests for the [AlinImobiliare website](https://github.com/4l1nU/AlinImobiliare-Website), implemented using **C#, Selenium WebDriver, and NUnit**.  
Designed to verify core functionality and ensure the website works as expected.

## Tested Features
- User registration, login, and logout  
- Property listing management (create, edit, delete)  
- Reviews submission  
- Online booking for apartment viewings  
- Search and filter functionality
